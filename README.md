# 🌿 Sistema de Monitoramento Ambiental - Irrigação Inteligente

## 📌 Sobre o Projeto
Este repositório contém o desenvolvimento do **Projeto Integrador** do [Coloque o Nome do Semestre, ex: 2º Semestre] da faculdade [Nome da Instituição]. 

O objetivo principal é criar um sistema de monitoramento ambiental focado em **irrigação inteligente**. O sistema utiliza sensores para medir a umidade do solo e o clima, processando esses dados para otimizar o uso da água de forma automatizada.

## 🛠️ Hardware Utilizado
* **Microcontrolador:** ESP32 (DevKit V1)
* **Sensores:** [Ex: Sensor de umidade do solo Higrômetro, DHT22 para temperatura/umidade]

## 💻 Tecnologias e Softwares
* **Python:** Utilizado para o desenvolvimento da inteligência do projeto, análise de dados e lógica de decisão.
* **C++/Arduino IDE:** (usando para programar o ESP32).
* **Git & GitHub:** Para controle de versão e documentação.
